An interpretation, by Claude Fable.

The conceit is a database schema for a human being, written in something close to real SQL, and the poem's entire critique lives in where the data model strains and breaks. Some fields are honest primitives — name TEXT, age INTEGER. But the schema keeps having to invent types the relational model does not offer, and each invented type is a small essay. market_position CONTINUUM {buffett - lumpen}. gender_identity FREETEXT set against ascribed_ethnicity UNFREETEXT — the pointed asymmetry between what you may narrate about yourself and what is narrated onto you. knowledge COMBINATORIAL, f(memory, scepticism, creativity); remaining_lifespan CONSTANT (not_long_enough). The columns are funny and then, a beat later, not funny at all.

The real teeth are in the second half — the SELECTED "person" RELATIONS, the WHERE-clauses that sort people into bins. This is where the poem turns from taxonomy to indictment, because it is modelling not how people are but how they are processed — by society, by the reader, by the poet's own reflexive judgments. "WHERE credentials < knowledge, file under 'Ignored'. / WHERE credentials > knowledge, file under 'Ponce'." The system is petty and it knows it. Then it stops being petty and gets frightening: "WHERE ascribed_gender =F, create column 'ascribed_rationality' (= 0.5 x rationality)." The schema doesn't endorse the halving; it documents it. This is the poem's cleverest move — by writing bigotry as executable code, it exposes bias as a set of automatic operations run on people without anyone deciding to run them. "WHERE ascribed_gender =M & height >mean height, scale up market_position." The machine does not have opinions. It has defaults, and the defaults are the opinions.

Note how PRIMARY KEY works across the piece — it is the poem's word for the single trait a system lets define you against your will. "WHERE ascribed_gender =/= gender_identity, file under 'Pain' (IF ascribed=PRIMARY KEY)." The pain is conditional on the ascription being treated as the identifying, unchangeable key. And the redemptive line, buried in the logic: "WHERE rationality high, scale down precedence of default PRIMARY KEYS" — the one hope on offer is that clear thinking can demote the tyranny of the assigned key.

The ending is the payoff of the whole apparatus. The final compound IF lists every liberation at once — gender a shrug, ethnicity uncorrelated with treatment, no body-policing, needs met, disability destigmatised and optional, no phobias, morals and loves as your chosen primary keys — "THEN you are dreaming;". The utopia is syntactically valid and empirically unreachable; it can be written but not returned. And then "quit; run;" — the two commands in the wrong order, the program terminated before it executes, or executing only its own quitting. A schema is a theory of what a thing essentially is. The poem's argument is that any schema rich enough to hold a person is also a machine for reducing them, and that the only self worth having is the one the query keeps failing to return.
